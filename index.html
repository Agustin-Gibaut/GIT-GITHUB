Aprende Git y GitHub

Git es la herramienta que gestiona el control de versiones del código fuente a nivel local y distribuido. GitHub es una plataforma en línea que utiliza Git para proporcionar funcionalidades adicionales de colaboración, gestión de proyectos y automatización.

  • ¿Cómo subir un proyecto?
  • ¿Cómo clonar un repositorio?

Pasos para subir un proyecto a GitHub

1. Inicializa un repositorio local: En tu terminal, navega al directorio de tu proyecto y ejecuta `git init` para inicializar un nuevo repositorio de Git.

2. Añade archivos al repositorio: Usa `git add .` para agregar todos los archivos del directorio al área de preparación. Puedes especificar archivos individuales si lo prefieres.

3. Realiza un commit: Ejecuta `git commit -m "Mensaje de commit"` para guardar tus cambios en el repositorio local con un mensaje descriptivo.

4. Conecta tu repositorio local a GitHub: Usa `git remote add origin ` para añadir el repositorio remoto de GitHub. Reemplaza `` con la URL de tu repositorio en GitHub.

5. Sube tus cambios a GitHub: Finalmente, ejecuta `git push -u origin master` para subir los cambios a tu repositorio en GitHub. Puedes necesitar ingresar tus credenciales de GitHub.

¿Cómo clonar un repositorio en Git?

1. Obtén la URL del repositorio: Ve al repositorio en GitHub y copia la URL del repositorio (generalmente terminando en `.git`).

2. Clona el repositorio: En tu terminal, navega al directorio donde quieres clonar el repositorio y ejecuta `git clone `. Esto creará una copia del repositorio en tu máquina local.

3. Navega al directorio del repositorio: Usa `cd ` para moverte al directorio recién creado que contiene el repositorio clonado.

4. Verifica los cambios: Puedes usar comandos como `git status` y `git log` para verificar el estado del repositorio y los commits.

5. Configura el repositorio: Si necesitas hacer cambios, configura tu repositorio con tus credenciales y ajustes necesarios.

6. Actualiza el repositorio: Usa `git pull` para obtener las últimas actualizaciones del repositorio remoto y mantener tu copia local sincronizada.
